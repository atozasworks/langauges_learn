# PWA Icons

This directory should contain the following icon files for the PWA:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Generating Icons

1. Open `generate-icons.html` in your browser
2. Click "Generate All Icons"
3. Click "Download All Icons" or download each icon individually
4. Save all icons in this `icons/` directory

## Alternative: Using Online Tools

You can also use online PWA icon generators:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Upload a 512x512 icon and it will generate all required sizes.

## Icon Requirements

- Format: PNG
- Purpose: any maskable (for Android)
- Transparent background recommended
- Square aspect ratio (1:1)
