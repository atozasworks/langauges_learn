# Professional Map App

A modern, professional map application built with MapLibre GL JS, OpenStreetMap tiles, and pure HTML/CSS/JS. Features place search, beautiful UI, and responsive design.

## Features
- Vector map rendering (MapLibre GL JS)
- Place search (OpenStreetMap Nominatim)
- Custom markers and popups
- Responsive, modern UI

## Getting Started
Just open `index.html` in your browser. No build step required!

## Customization
- Change the default center in `main.js`.
- Update the style or add more controls as needed.
